# Day 7: Exploratory Data Analysis & Sprint 1 Review

## Objective
The goal for today is to practice EDA (Exploratory Data Analysis) by plotting distributions and identifying patterns/outliers, and to wrap up the first sprint of our 60-days data science journey.

- Wrote a Sprint 1 summary for LinkedIn.
