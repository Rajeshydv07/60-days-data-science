# 60 Days of Data Science 🚀

This repository tracks my journey through the 60 Days of Data Science challenge. Each day focuses on a new concept or project in the field of Data Science.

## Project Overview: Sales Analysis (Superstore)

**Domain:** Sales (Retail)  
**Dataset:** [Kaggle - Superstore Sales Dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)

### Problem Statement
The goal of this project is to perform an in-depth Exploratory Data Analysis (EDA) on the Superstore Sales dataset. We aim to:
1. Identify historical sales trends and seasonality.
2. Determine which product categories and sub-categories are the most profitable.
3. Analyze regional performance to identify underperforming markets.
4. Provide actionable business insights to help optimize inventory and marketing strategies.
