# Day 5: Data Ingestion and Preliminary Exploratory Data Analysis (EDA)

## Project Overview
This repository documents a 60-day intensive journey into Data Science. On Day 5, the primary focus is to establish a robust foundation for the project by ingesting the raw dataset and conducting preliminary exploratory data analysis.

## Domain & Dataset
- **Domain:** Retail & Sales Optimization
- **Dataset Source:** Kaggle (Sales Forecasting - Superstore Dataset)

## Objectives Achieved
1. **Data Ingestion:** Successfully implemented data loading pipelines utilizing the `pandas` library to ingest the local `.csv` dataset.
2. **Structural Inspection:** Analyzed the underlying structure of the dataset by extracting its dimensionality (`shape`) and feature attributes (`columns`).
3. **Data Sampling:** Extracted representative samples (`head()`) to visually validate data integrity and formatting.
4. **Target Variable Identification:** Formally defined the target variable, **`Sales`**, establishing the predictive objective for subsequent machine learning models.

## Repository Structure
- `day5_data_analysis.ipynb`: The core analytical Jupyter Notebook containing the structural data analysis.
