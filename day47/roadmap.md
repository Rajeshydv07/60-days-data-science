# Capstone Project Roadmap (Days 47–60)

This roadmap outlines the systematic execution of the Customer Intelligence Platform from initial design to final deployment.

## Phase 1: Foundation & Data Engineering (Days 47-49)
- **Day 47**: Project Kickoff, Architecture Design, and Requirement Gathering (Current).
- **Day 48**: Data Sourcing, Exploratory Data Analysis (EDA), and Data Cleaning.
- **Day 49**: Feature Engineering, RFM (Recency, Frequency, Monetary) modeling, and setting up the Feature Store.

## Phase 2: Core Machine Learning Models (Days 50-53)
- **Day 50**: Customer Segmentation Model (K-Means Clustering).
- **Day 51**: Churn Prediction Model (Data prep, addressing class imbalance, Baseline Model).
- **Day 52**: Churn Prediction Optimization (Hyperparameter tuning, XGBoost, Model Evaluation).
- **Day 53**: Time-Series Forecasting (CLV prediction and revenue forecasting using Prophet).

## Phase 3: API Development & Integration (Days 54-55)
- **Day 54**: Building the Model Serving Layer with FastAPI.
- **Day 55**: API Testing, caching strategies, and documentation (Swagger/OpenAPI).

## Phase 4: Frontend Dashboard & Visualization (Days 56-57)
- **Day 56**: Setting up the Interactive Dashboard (Streamlit or Dash).
- **Day 57**: Connecting Frontend to API endpoints, creating visualizations for churn, segments, and revenue.

## Phase 5: Deployment & CI/CD (Days 58-59)
- **Day 58**: Containerization with Docker and writing deployment scripts.
- **Day 59**: Cloud Deployment (AWS/GCP), setting up basic CI/CD pipelines, and production monitoring.

## Phase 6: Wrap-up & Presentation (Day 60)
- **Day 60**: Final testing, preparing the presentation, updating GitHub README, and LinkedIn reflection.
